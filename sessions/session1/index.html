Your Mission:

  • Create index.html
    • Add a title
    • Add a meta tag saying you are the author
    • Add a page title using header tag
    • Add some content with paragraph tags, your name and favorite food!
  • Create stylesheet (make sure to include it!)
    • Add div tags and add id attribute
    • Add id attributes to header tags and paragraphs
    • Set default background color, font size, and font style
    • Set background color of your choosing to the page in the stylesheet by using the id attributes

That's it! There's the essence of most webpages, a backbone that shows content and a way to make it all look good.
